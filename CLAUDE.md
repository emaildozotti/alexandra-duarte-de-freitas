# alexandra-duarte-de-freitas LP

Repo standalone da Landing Page de Alexandra Duarte de Freitas.
Pipeline v9 — Terapeuta10x.

## Stack
- React 19 + Vite 6 + Tailwind v4 + Framer Motion v12
- Fontes: Lora (display) + Source Sans 3 (sans) via @fontsource

## Deploy
- Push para `main` dispara deploy automatico na Vercel
- Para editar: alterar arquivos em `src/` → `npm run build` → `git push`

## Fotos
- `public/foto-hero.jpg` — foto principal (3:4)
- `public/foto-about.jpg` — foto da secao Sobre (3:4)
- Enquanto nao tiver foto, o fallback mostra iniciais + nome
